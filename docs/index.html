What it does

Three things, done well.

No bloat, no cloud detours. Flow does one job — turning speech into text inside any app — and tries to do it without surprises.

On-device transcription

MLX-Whisper Large v3 Turbo runs locally on your Apple Silicon Mac. No upload, no streaming to a server — your voice never leaves the device.

Three cleanup modes

Verbatim for the raw transcript, Formal for polished prose, Anti-Wut to de-escalate heated messages. Switch with a modifier key mid-tap.

Works in any app

Flow types the result wherever your cursor is — email, Slack, IDE, browser, terminal. No integrations, no plugins. It just pastes.

Who it's for

Built for people who think out loud.

Privacy-conscious

Privacy & security

Privacy by default.

Flow is built so that even the developer can't see your dictations. Here's exactly what happens — and what doesn't.

  • Audio never leaves your Mac

    MLX-Whisper runs entirely on Apple Silicon. The recorded audio buffer is held in memory, transcribed, then discarded. No upload, no streaming, no cache.

  • Cleanup is opt-in

    Verbatim mode never contacts Groq. Formal/Anti-Wut send only the transcribed text — not the audio — and only if you've entered an API key.

  • History stays on your disk

    Past dictations live in ~/.worknetic-flow/history.json. Plain JSON, easy to inspect, easy to delete.

  • No telemetry

    No analytics, no crash reporting that phones home, no auto-update check. Flow knows nothing about how often you use it.

  • MIT — audit the source

    Every line is public on github.com/alpeppo/flow. The install script is 130 lines of plain bash — read it before running it.

  • Community-driven

    Flow is built by volunteers, for the community. No company, no investors, no monetization plan. Issues and pull requests welcome.

Install

One Terminal command.

No installer wizard, no Apple-Developer detour. Paste, press Enter, done.

Why not just a DMG?

Apple's Gatekeeper blocks unsigned apps. Notarization costs USD 99/year for a developer account, which we'd rather not pay for an open-source project. The Terminal command sidesteps this cleanly: curl downloads don't get the Gatekeeper quarantine flag, and the script explicitly strips it from the installed app. You can read the script first — 130 lines, no root, no kernel extensions.

1Run the command

Open Terminal.app (Cmd+Space, type "Terminal", hit Enter). Paste this line, press Enter:

$ curl -fsSL https://alpeppo.github.io/flow/install.sh | bash

The script will show every step live: system check, download, mount, install, permissions hint. About 60 seconds.

2Grant permissions

Flow launches automatically. macOS will ask for two permissions:

  • Microphone — system dialog on first recording. Allow.
  • Accessibility — System Settings → Privacy & Security → Accessibility. Toggle Flow on. Required so Flow can detect the global hotkey and paste into focused apps.

3Add your Groq key (optional)

Without a key, you get Verbatim mode — the raw Whisper transcript. That works for most cases.

With a free Groq API key, you unlock:

  • Formal (Ctrl+fn) — Llama rewrites your speech as polished prose.
  • Anti-Wut (Shift+fn) — de-escalates emotionally-charged dictations before paste.

Paste the key under Einstellungen → Speichern.

4Try it

  • Start recording: double-tap fn. The pill appears at the bottom of your screen.
  • Stop & paste: double-tap fn again. Your transcript lands in the focused app.
  • Cancel: ESC or click the ✕ on the pill. No transcription, no paste.

Open the main window (history, dictionary, commands, settings) from the menubar icon or by clicking Flow in your Dock.

FAQ

Frequently asked.

Is any audio sent to a server?

No, never. Whisper runs on-device via MLX. Even when you use Formal or Anti-Wut cleanup, only the transcribed text is sent to Groq — the audio buffer never leaves your Mac.

Why would I need a Groq API key?

Only if you want Formal or Anti-Wut cleanup. Verbatim mode (the default) works completely offline without any key. Groq has a generous free tier that's more than enough for personal use.

What are the system requirements?

macOS 13 (Ventura) or newer, on Apple Silicon (M1, M2, M3, M4). Intel Macs aren't supported because MLX-Whisper needs the Metal backend that ships with Apple Silicon.

Why isn't Flow signed and notarized?

Apple charges USD 99 per year for a developer account. Flow is a community project with zero revenue — we'd rather not pay that out of pocket. The Terminal installer is a clean workaround: curl downloads aren't quarantined by Gatekeeper, so the install just works.

How do I uninstall Flow?

Drag /Applications/Flow.app to the Trash. For a clean removal, also delete the config & history at ~/.worknetic-flow/.

How do I report a bug or request a feature?

Open an issue on GitHub. PRs welcome — for bigger changes, open an issue first so we can align on the approach.
